How it works

Agent가 직접 DB를 보는 것이 아니라, 승인된 도구만 호출합니다.

ArchDox는 Agent에게 내부 DB, provider API key, 원본 저장소 접근권을 주지 않습니다. Agent는 등록된 tool/action만 요청하고, ArchDox가 인증, 권한, quota, 정책을 확인한 뒤 Engine Boundary를 실행합니다.

1

가입 또는 로그인

개인 또는 사무소 계정으로 ArchDox Engine 사용 권한을 확인합니다.

2

Agent 선택

ChatGPT, Claude, Cursor, Codex, Custom Agent 중 연결할 채널을 선택합니다.

3

도구 범위 승인

감리일지 검토, 법령 변경 조회, 체크리스트 생성 등 허용할 tool pack을 승인합니다.

4

검토 실행

Agent 요청은 ArchDox Engine API/MCP Gateway를 통해 구조화된 finding과 action으로 반환됩니다.
