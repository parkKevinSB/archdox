@archdox/engine

const review = await archdox.review.create({
  documentType: "CONSTRUCTION_DAILY_LOG",
  effectiveDate: "2026-06-04",
  context: {
    workType: "FOUNDATION_CONCRETE_PLACEMENT",
    structureType: "REINFORCED_CONCRETE",
    site: "초읍동 353-1"
  }
})

const result = await review.run({
  checks: ["rules", "evidence", "wording"],
  output: "structured-findings"
})

// Every result is source-backed and audit-ready
console.log(result.findings[0])

Why ArchDox

일반 AI 호출과 믿을 만한 업무 엔진의 차이

01

문서를 검토 컨텍스트로 변환

공종, 구조, 현장 조건 같은 표현을 표준 코드와 업무 의미로 정규화합니다.

02

기준일과 룰셋 고정

어떤 날짜, 어떤 법령 버전, 어떤 룰셋으로 검토했는지 결과에 남깁니다.

03

수정 가능한 결과

어느 섹션을 무엇으로 보완해야 하는지 구체적인 액션으로 제공합니다.
