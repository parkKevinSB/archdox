ArchDox Developers

외부 AI와 업무 시스템을 ArchDox Engine에 안전하게 연결합니다.

ArchDox Developers는 Codex, Claude, Cursor, 사내 Agent, 업무 시스템이 ArchDox의 문서 검토, 법령 근거 조회, 감리 업무 컨텍스트 검증 기능을 같은 경계 안에서 호출하도록 돕는 공개 가이드입니다.

MCP 연결 가이드 보기 로그인 후 키 발급

Boundary

문서는 공개 가이드에, 키와 사용량은 로그인 앱에 둡니다.

Public

공개 개발자 문서

ArchDox MCP가 무엇인지, 어떤 tool이 있는지, Codex/Claude/Cursor에 어떻게 연결하는지 설명합니다. API Key 원문이나 고객 데이터는 공개 페이지에 두지 않습니다.

App

로그인 후 Developer 화면

실제 Engine API Key 발급, MCP config 생성, 연결 테스트, 사용량 확인은 app.archdox.co.kr/developer에서 처리합니다.

Admin

플랫폼 운영 화면

Admin은 전체 키, quota, usage, 실패 호출, tool catalog를 운영자가 관측하는 곳입니다. 일반 사용자의 자기 연결 화면과 섞지 않습니다.

Products

처음 열리는 개발자 기능은 MCP Gateway입니다.

MCP Gateway AI 도구가 ArchDox 문서 검토, 법령 업데이트, 법령 검색 tool을 호출합니다. Engine API Key 로그인한 사용자가 자기 키를 발급하고 폐기하며 사용량을 확인합니다. Legal Updates ArchDox가 동기화한 법령 변경 digest를 사용자와 외부 Agent가 같은 기준으로 읽습니다.

Start

연결은 공개 문서를 읽고, 실제 키는 로그인 후 발급하세요.

API Key는 한 번만 표시됩니다. 연결 후에는 Developer 화면에서 live smoke로 실제 MCP 호출을 확인할 수 있습니다.
